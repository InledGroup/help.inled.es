# Bienvenido a Insuite Office

Esta es una página de ejemplo creada **localmente** para que puedas ver el diseño ahora mismo sin depender de GitHub.

### ¿Qué puedes hacer con Insuite Office?
*   **Edición en tiempo real**: Todos tus cambios se guardan localmente.
*   **Privacidad total**: Tus documentos nunca salen de tu navegador.
*   **Compatibilidad**: Funciona en cualquier dispositivo moderno.

### Cómo funciona este sistema
El sistema está diseñado para leer tanto de tus **GitHub Issues** como de archivos locales en la carpeta `src/content/help/`. Esto te permite:

1.  Probar cambios de diseño rápidamente en local.
2.  Tener una documentación base que siempre esté disponible.
3.  Actualizar dinámicamente desde la nube mediante agentes de IA.
